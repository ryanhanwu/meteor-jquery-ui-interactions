# Meteor jQuery UI Interactions

## About

This is a [meteor](https://www.meteor.com/) package which contains jQueryUI's interaction libries as below

* Draggable
* Droppable
* Resizable
* Selectable
* Sortable


## Installation
```
meteor add ryanwu:jquery-ui-interactions
```
